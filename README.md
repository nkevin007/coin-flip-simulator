This is a virtual coin flip simulator built with React + TypeScript + Tailwind CSS.

## Features

- Flip 5 / 10 / 30 / 100 coins
- See how many heads/tails and their percentages
- Educational Law of Large Numbers explanation

## How to Run

1. Install dependencies  
```bash
npm install
```

2. Start local server  
```bash
npm run dev
```

3. To build for production  
```bash
npm run build
```
